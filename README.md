# 365 Days of Code

This repository documents my long-term software development practice,
focused on consistency, clean code, and professional engineering habits.

## Practice Schedule
- Mandatory sessions: Monday, Wednesday, Friday
- Optional bonus sessions: Tuesday, Thursday

## Goals
- Strengthen programming fundamentals
- Improve problem-solving skills
- Apply clean code and best practices
- Build a solid public learning record

## Technologies
- Java
- Git

## Organization
Content is organized by topic rather than by day.
Each session is documented through code comments and a learning log.
